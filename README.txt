GÉOBREVET
-------------------------------
NOTES DE VERSION
2016.04.1.0
-------------------------------

Date de publication : 3 avril 2016


Nouveautés de la version 2016.04.1.0


Dans la version 2016.04.1.0 de GÉOBREVET, nous avons créé ou amélioré les fonctionnalités suivantes :

	- Corrections orthographiques sur la fiche "Repères 3ème" (Histoire).
	- Ajout des cartes 3ème dans le module d'apprentissage "Géographie".
	- Mise totale en conformité du logiciel avec les programmes nationaux.


NB : si vous souhaitez voir des fonctions particulières voir le jour, vous pouvez contacter le Support pour formuler votre demande. Bien, sûr, d’autres fonctions sont prévues pour une version ultérieure.
